# Lucky Draw Game

一个功能丰富的在线抽奖游戏平台，包含用户管理、多种抽奖模式、客服系统和完整的管理后台。

## 快速开始

### 自动部署（推荐）
```bash
chmod +x deploy.sh
./deploy.sh
```

### 手动部署
详细部署说明请查看 [部署指南](DEPLOYMENT_GUIDE.md)

## 核心功能

- 🎮 **多种抽奖模式** - 支持等多种玩法
- 👥 **用户管理系统** - 完整的注册、登录、个人信息管理
- 🎁 **物品系统** - 物品获得、展示、分解功能
- 💰 **充值系统** - 多种充值方式，金币管理
- 📝 **签到系统** - 每日签到奖励机制
- 💬 **客服系统** - 实时客服分配和聊天功能
- 🛠️ **管理后台** - 完整的后台管理界面
- 🔐 **权限控制** - 基于角色的访问控制
- 📊 **数据统计** - 详细的运营数据统计
- 🎨 **主题管理** - 可自定义的界面主题

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **后端**: PHP 7.4+
- **数据库**: MySQL 5.7+ / MariaDB 10.3+
- **Web服务器**: Apache / Nginx

## 系统要求

- PHP 7.4+ (推荐 8.0+)
- MySQL 5.7+ 或 MariaDB 10.3+
- Web 服务器支持 URL 重写
- 2GB+ RAM, 10GB+ 存储空间

## 快速配置

1. **创建数据库**
   ```sql
   CREATE DATABASE lucky_draw CHARACTER SET utf8mb4;
   ```

2. **配置数据库连接**
   编辑 `server/config/database.php`

3. **设置文件权限**
   ```bash
   chmod -R 755 ./
   chmod -R 777 images/ uploads/
   ```

4. **创建管理员**
   访问 `create-super-admin.html`
