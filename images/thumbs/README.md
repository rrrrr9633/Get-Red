# Lucky页面小图片目录

此目录用于存储Lucky页面在main.html中显示的小图片。

## 文件命名规则
- 文件名应与Lucky页面文件名对应（不含.html后缀）
- 例如：lucky1.html 对应 lucky1.jpg/lucky1.png
- 支持的格式：JPG、PNG、GIF、WebP

## 图片规格建议
- 建议尺寸：120x120px
- 文件大小：不超过1MB
- 建议使用方形图片以获得最佳显示效果

## 使用方法
1. 在管理后台选择"奖品管理"
2. 点击"设置小图片"按钮
3. 选择对应的Lucky页面
4. 上传符合规格的图片
5. 小图片将在主页面的game-grid中显示
