# 商城图片资源

此文件夹用于存放商城各分类的展示图片。

## 图片规格建议

- **尺寸**: 200x200px 或更大（保持正方形）
- **格式**: PNG（支持透明背景）
- **文件大小**: 建议小于 500KB

## 图片文件命名

- `gold-escort.png` - 金牌护航图片
- `knife-exchange.png` - 1:1跑刀图片
- `skin-exchange.png` - 皮肤兑换图片
- `legendary-exchange.png` - 传说级兑换图片

## 当前状态

如果图片文件不存在，将显示默认的 emoji 图标：
- 🛡️ 金牌护航
- 🔪 1:1跑刀
- 🎨 皮肤兑换
- ⭐ 传说级兑换

## 使用方法

1. 将对应的图片文件放入此文件夹
2. 确保文件名与上述命名一致
3. 刷新页面即可看到自定义图片
